# Takumi Farm Operations

A lightweight operations management platform for Takumi Farm.

The system is designed to help drivers quickly log vehicle activities while providing management with visibility into daily operations.

## Planned Stack

- Frontend: React + TypeScript + TailwindCSS
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- Deployment: Ubuntu + Nginx + Gunicorn
